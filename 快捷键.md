# 快捷键

Ctrl + C：复制                                                   Ctrl + V：粘贴
Ctrl + A：全选 	                                           Ctrl + X：剪切
Ctrl + Z：撤销                                                    Ctrl + S：保存
Alt + F4：关闭窗口                                            Shift+ Del：永久删除
win+R：命令行                                                  win+E：打开我的电脑
Ctrl + Shift + Esc：打开任务管理器(注：explorer可以重启桌面)
win + Tab：切换程序
