# Markdown学习

## 标题

“一级标题：# + 标题内容”

“二级标题：## + 标题内容”

...

"六级标题：###### + 标题内容"

## 字体

*“    * + 文本 + *    ”： 斜体*

**“    ** + 文本 + **    ”： 黑体**

***" 三个 *  + 文本 + 三个 * "  ： 黑体+ 斜体***

~~“    ~~ + 文本 + ~~    ”： 删除体~~

## 引用

>  " > + 要引用的内容 "



## 分割线

---

***

“---/3个* ”

## 列表

“ 1 + . + 空格 ”： 有序列表

1. A
2. B
3. C

" -  + 空格": 无序列表 “-”号等同于 “ * ”/“ + ”

- A
- B
- C

"- [ ] text"   

- [ ] 待完成

"- [x] text"（注意[]里面的是英文字符x, 而不是对错的错号）

- [x] 已完成
